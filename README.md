# Brute-force Numeric List

## 📄 Description

This file contains a numeric wordlist generated from **0000 to 9999**.
Each number is zero-padded to maintain a fixed length of 4 digits.

## 📌 Usage

This wordlist can be used for:

* Testing password strength
* Learning about authentication systems
* Security research in controlled environments

## ⚙️ Format

* Total entries: 10,000
* Range: 0000 → 9999
* Each entry is in a new line

Example:
0000
0001
0002
...
9999
